Networking Fundamentals – IPv4 Subnetting

-Overview

This session focused on IPv4 subnetting and how subnet masks are used to divide networks into smaller logical segments. Subnetting allows administrators and security professionals to organize networks efficiently and control communication between devices.

-Topics Covered

* IPv4 address structure
* Binary representation of IP addresses
* Subnet mask calculations
* Determining network addresses
* Determining broadcast addresses
* Calculating valid host ranges

-Why Subnetting Matters in Cybersecurity

Subnetting is important in cybersecurity because it allows professionals to:

* Understand network segmentation
* Identify devices within specific network ranges
* Perform accurate network reconnaissance
* Analyze suspicious traffic across subnets
* Support incident response investigations

-Example Concept

Given an IP address and subnet mask, subnetting allows us to determine:

* Network address
* Broadcast address
* Total number of hosts
* Usable host range

-Key Takeaway

Understanding subnetting is critical for networking and cybersecurity professionals because it provides the foundation for analyzing how devices communicate within a network.
